## Agenda

Agenda is an AI-powered calendar app — add events, set reminders, track habit streaks, and plan your week just by chatting with it. Built on Angular and AWS Amplify.

## Features

- **AI chat assistant**: Create, reschedule, or delete events, get planning advice, and set reminders through natural conversation (Amazon Bedrock / Claude).
- **Habit streaks**: Track daily habits with goals, deadlines, and progress history.
- **Authentication**: Secure sign-in via Amazon Cognito.
- **API**: GraphQL endpoint with AWS AppSync.
- **Database**: Real-time database powered by Amazon DynamoDB.
